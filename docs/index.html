Music Commands:

Command Description Usage Aliases
play Plays music from youtube or given url. if there are no arguments it resumes the player ^play <url/words> p
join Joins a voice channel ^join
stop Stops the music player ^stop
pause Pauses the currently playing track ^pause
resume Resumes the track if paused ^resume unpause
queue Shows the queue ^queue q
clear Clears the queue ^clear
seek Seeks to a point in the track ^seek [hh:[mm:[ss]]]
export Exports the playlist to hastebin or wastebin ^export
shuffle Shuffles the queue ^shuffle
skip Skips the currently playing song ^skip <number of songs to skip>
np Shows the currently playing song ^np nowplaying | playing
