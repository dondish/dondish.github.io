Music Commands:

Command Description Usage Aliases
play Plays music from youtube or given url. if there are no arguments it resumes the player ^play <url/words> p
join Joins a voice channel ^join
stop Stops the music player ^stop
pause Pauses the currently playing track ^pause
resume Resumes the track if paused ^resume unpause
queue Shows the queue ^queue q
clear Clears the queue ^clear
seek Seeks to a point in the track ^seek [hh:[mm:[ss]]]
export Exports the playlist to hastebin or wastebin ^export
shuffle Shuffles the queue ^shuffle
skip Skips the currently playing song ^skip <number of songs to skip>
np Shows the currently playing song ^np nowplaying | playing
repeat Toggles between repeat modes ^repeat <mode>
reverse Reverses the entire queue ^reverse
history Shows the history of played songs ^history
remove Remove a song from the queue at the given index ^remove <index>
back Returns one song backwards in the queue ^back

Moderation Commands:

Command Description Usage Aliases
settings Manages guild settings ^settings <options> <optionArguments>
welcomer Enable or disable welcomer ^welcomer
purge Purges an amount of messages in the channel ^purge <amount>
mute Mutes a member so their messages will be deleted. 1 day by default ^mute <user> <timeInMinutes> <reason>
unmute Unmutes a member ^unmute <user> <reason>
softban Bans and then unbans a user to clear his messages. ^softban <user> <reason>
warn Warning based commands ^warn <@user mention/amount @user mention/clear @user mention>
